# Character Reference Guide

このフォルダは、漫画画像で登場する「娘」と「父」の見た目を固定するための参照資料を置く。

## 娘

- 5〜7歳くらい
- 黒〜こげ茶のボブ、前髪あり、少し内巻き
- 大きめの目、好奇心が強い表情
- 黄色のパーカー
- 左側に小さな星形ヘアピン
- 明るく、「なんで？」と聞く雰囲気

## 父

- 30代後半〜40代前半
- 黒〜こげ茶の短髪、自然な癖毛
- やさしい目、軽い無精ひげ、丸眼鏡
- ネイビーのカーディガン、白Tシャツ
- ノートPC、マグカップ、メモ帳が似合う
- 学び直している父親、少しぼやく雰囲気

## Shared Style

- 日本語ブログ向けのやさしい漫画イラスト
- 2D digital manga / web comic style
- 表情が読みやすい
- ダークテーマの記事上でも見やすい配色
- 背景はシンプルな室内、机、ノートPC、本棚など
- 宇宙・科学・プログラミング要素は小物として控えめに入れる

## Base Prompt

```text
Create a consistent character reference illustration for a Japanese educational father-daughter blog.
Style: warm 2D digital manga / web comic illustration, clean line art, soft but readable colors, expressive faces, simple indoor study room background, suitable for a dark themed blog.

Daughter: 5 to 7 years old, small childlike silhouette, black to dark brown bob haircut with bangs and slightly inward curled ends, large curious eyes, yellow hoodie, light inner shirt, small star-shaped hairpin on the left side, cheerful and inquisitive expression.

Father: late 30s to early 40s, average build, black to dark brown short slightly wavy hair, gentle eyes, light stubble, round glasses, navy cardigan over a white T-shirt, kind but slightly tired father expression, learning with a laptop and notes.

Keep the character designs stable and recognizable. No text, no logos, no watermark.
```

## Negative Prompt

```text
Do not change the daughter's bob haircut, yellow hoodie, or star-shaped hairpin.
Do not remove the father's round glasses, navy cardigan, or light stubble.
Avoid photorealism, overly flashy anime style, complex backgrounds, unreadable text, random letters, logos, or watermarks.
```
